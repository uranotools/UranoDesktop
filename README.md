## 🌟 ¿Qué es Urano Desktop?
Urano Desktop es mucho más que un simple chat de IA. Es una plataforma donde puedes construir y personalizar tus propios asistentes locales. A diferencia de otros sistemas, Urano vive en tu computadora, respetando tu privacidad y permitiéndote extender sus capacidades hasta donde tu imaginación llegue.

## 🧠 Personalización al Máximo

### 1. Configuración de Agentes
En el **Dashboard de Agentes**, puedes dar vida a múltiples personalidades. Cada agente puede ser configurado con:
- **Nombre y Emoji**: Para identificarlos visualmente.
- **Sistema de Prompt (Persona)**: Define su comportamiento (ej. "Eres un experto contador" o "Eres un programador senior").
- **Modelo de IA**: Elige entre modelos potentes en la nube (OpenAI, Anthropic) o modelos totalmente privados ejecutándose en tu PC a través de **Ollama**.

### 2. Módulos MCP (Model Context Protocol)
Urano utiliza el estándar MCP para que tus agentes tengan "manos" y puedan interactuar con el mundo real. Desde el **McpManager**, puedes conectar integraciones de primer nivel:
- **Google Workspace**: Maneja tu **Calendar**, busca en **Drive**, lee tu **Gmail** o edita **Docs/Sheets**.
- **Herramientas de Desarrollador**: Conéctate a **GitHub** o utiliza el sistema de búsqueda técnica.
- **Bóveda de Seguridad (Vault)**: Tus API Keys y credenciales están cifradas localmente con seguridad de grado militar.

### 3. Native Skills (.md)
¿Quieres enseñarle algo específico a un agente sin programar? Las **Skills** son archivos markdown simples que puedes crear o importar.
- Define protocolos de venta.
- Inyecta manuales de usuario de tus propios productos.
- Establece reglas de negocio específicas.
*El agente leerá estas instrucciones solo cuando sean relevantes, ahorrando memoria y mejorando la precisión.*
